# ホームページ

_ホーム_ ページは、Adobe Journey Optimizer B2B Primeの起動パッドです。 左側のナビゲーションの上部にある「_ホーム_」アイコンを選択して開きます。 **ようこそ[name]様、今日から何を作れますか？** ワンクリックでアイデアからアクションへと導くように設計されています。

![ ホームページ ](./assets/home-page.png){width="800" zoomable="yes"}

## 入力ボックス

大きな&#x200B;_質問するか、タスクを開始する_&#x200B;入力がホームページの中央に表示されます。 ここに何かを入力してAI対話型パネルの操作を開始するか、_添付_ （**+**）アイコンを使用して、送信する前にファイル（キャンペーン概要やCSVなど）を追加します。 これは、マーケティング管理全体で利用できるAI アシスタントと同じですが、ここで始めると新しい会話が開きます。

## インテントボタン

入力ボックスの下に、4つのクイックスタートボタンが表示されます。

| ボタン | 使用方法 |
|--------|------------|
| **プラン** | 戦略の策定：施策、プログラム、オーディエンスを構築する前に計画を立てます |
| **作成** | プログラム、ジャーニー、メール、人物リストなどを構築します。 |
| **測定** | レポート、エンゲージメント、送信時間の結果などのパフォーマンスを確認します。 |
| **最適化** | 運用中のコンテンツ（トラフィック制御、スコアリング、パーソナライゼーション）を改善する。 |

## クイックアクセス

作業空間にすばやく直接ジャンプするために使用できる4つのショートカットカードがあります。

| カード | アクセスできること |
|------|-----------------|
| **マーケティング** | プログラム、ジャーニー、オーディエンス（マーケティング管理）: |
| **Assets** | 電子メール、ページ、テンプレート、トークン： |
| **レポート** | パフォーマンスとエンゲージメントの指標： |
| **宛先** | ダウンストリームのシステムと同期： |

## スキル

_スキル_ カードの行には、すぐに起動できる価値の高いAI ワークフローが表示されます。

- **ペルソナベースのオーディエンスを作成** — ペルソナのシグナルと企業特性に基づいてグループ化されたセグメントを生成し、プログラムに参加する準備を整えます。
- **オムニチャネルプログラムの作成** – 電子メール、web、ダウンストリームの宛先にまたがるプログラムを1つの概要から作成します。
- **詳細にパーソナライズされた電子メールを作成** — トークンと動的コンテンツブロックを使用して、受信者ごとにパーソナライズされたブランドに即した電子メールのドラフトを作成します。
- **適合およびインテントベースのジャーニーのトラフィック制御** — アカウントの適合およびライブ購入者のインテントシグナルに基づいて、ジャーニーをまたいで連絡先をルーティングします。

そのワークフローに適した会話を開始するには、スキルカードをクリックします。 スキルは、チャットパネル内で利用できるものと同じ構成要素です。カードは、発見可能なエントリーポイントです。
