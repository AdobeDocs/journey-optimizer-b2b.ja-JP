# ジャーニーノード

ジャーニー（アカウントジャーニーまたはユーザージャーニー）を作成したら、オーディエンスを追加し、ノードを使用してジャーニーを構築します。 ジャーニーマップはキャンバスを提供し、複数手順の B2B マーケティングのユースケースを作成できます。

オーディエンスノードは、自動的にジャーニーの最初のノードになります。

* [アカウントオーディエンス](./account-audience-nodes.md)
* [ 人物オーディエンス ](./person-audience-nodes.md) （Beta）

様々なアクション、イベント、オーケストレーションのノードを複数の手順からなるクロスチャネルシナリオとして組み合わせることで、ジャーニーを構築します。 ジャーニーの各ノードは、論理パスに沿ったステップを表します。 次のノードタイプを使用して、アカウントジャーニーを構築します。

* [アクションの実行](./action-nodes.md)
* [イベントのリッスン](./listen-for-event-nodes.md)
* [パスを分割](./split-merge-paths-nodes.md)
* [待機](./wait-nodes.md)
* [パスを結合](./split-merge-paths-nodes.md)
