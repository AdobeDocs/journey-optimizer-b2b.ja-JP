# Journey Optimizer B2B editionの AI アシスタントの質問ガイダンス

Journey Optimizer B2B editionで AI アシスタントをクエリする際の、次の質問例を確認してください。 この情報には、質問にフレーズを使用して AI アシスタントから最適な回答を得る方法に関するヒントも含まれています。

## 目標ベースの質問

次の質問の例は、AI アシスタントを使用する際に達成できる目標に従ってグループ化されています。

## 質問の言葉遣い

可能な限り正確に回答を得るには、質問を AI Assistant に明確かつコンテキストでフレーズ化する必要があります。 コンテキストを使用して明確な質問をする方法のガイダンスについては、次のヒントのリストを参照してください。

* タスクや質問を簡潔に述べます。
* あいまいな言語や複雑すぎる構文を避けて、理解を容易にします。
* AI アシスタントがより関連性の高い回答を生成できるように、タスクや質問に関する関連性の高いコンテキストを提供します。

次の表に、AI アシスタントを使用する際に従うことのできるベストプラクティスをいくつか示します。

| 実行 | 例 |
| --- | --- |
| <li>取得または分析するオブジェクトや情報を具体的に指定します。 <li>データオブジェクト名を引用符で囲んでみてください。 <li>オブジェクト名の一部しか知らない場合は、それを質問で指定することもできます。 | <li>「B2B アカウント」スキーマを使用するデータセットはどれですか？ <li>名前に「アカウント」が含まれている、アクティブ化されたオーディエンスを表示します。 メンバー数別にランク付けします。 |
| <li>曖昧さを避け、明確な言語を使用する。 <li>クエリをより明確にするために、正確な用語を使用します。 <li>Adobe Experience PlatformとAdobe Journey Optimizer B2B editionに関する質問をする場合は、Experience PlatformまたはAdobe Journey Optimizer B2B editionに固有の用語を使用して、回答の関連性を高めるようにします。 | <li>「アカウントのオーディエンス」のメンバーは何人ですか？ <li>アカウントジャーニーで「自分のアカウントオーディエンス」アカウントオーディエンスを使用するのはいくつですか？ |
| <li>結果を絞り込むためのコンテキストの提供または条件の指定。 <li>質問にフィルター条件を使用して、応答内のデータ量を制限します。 | <li>アクティブ化されておらず、6 か月以上前に作成されたのみで変更されていないアカウントオーディエンスを表示します。 <li>過去 7 日間に公開されたアカウントジャーニーを表示し、1,000 人を超えるメンバーを持つアカウントオーディエンスを使用する |

| Don&#39;t | 例 |
| --- | --- |
| あいまいな言語やあいまいな言語を使用します。 | <li>データセットに関する情報を提供する <li>journey x の機能 <li>「ACME オーディエンス」には何人のユーザーがいますか？ <li>セグメントを表示します。 <li>リスト属性。 |
| 不完全なリクエストをおこなう。 | <li>&quot;Luma - ロイヤルティデータセット&quot; |
| コンテキストのない知識を想定します。 | <li>過去 6 か月間のオーディエンス。 <li>クエリを作成します。 <li>ジャーニーを作成 |
| 過度に複雑なクエリを作成する。 | <li>すべてのオブジェクトとその依存関係にわたるデータ系列の包括的な分析を提供します。 |
| 条件またはパラメーターを省略します。 | <li>データセットの表示。 |

## サポートされていない質問の例

以下のリストには、Journey Optimizer B2B editionの AI アシスタントが現在サポートしていない質問の例が含まれています。

* 条件内で… フィールドグループの workEmail.address フィールドを使用するのは、どのアカウントオーディエンスですか？ 
* アカウントオーディエンスを使用したアクティブなジャーニーの数を表示します。サイズは、10,000 件を超え、5,000～10,000 件、1000～5,000 件、1,000 件未満のジャーニーを配信用のビジュアルで表示します
* アカウントジャーニー x を最後に更新したのは誰ですか？
* ソリューションの関心 x に購入グループメンバーを追加したアクティブなジャーニーの数。
* ソリューションの関心 x に購入グループメンバーを追加するアクティブなジャーニーは何ですか？
* 購入グループテンプレートの最も一般的な意思決定者タイトルは何ですか？
* グループテンプレート x を購入するための意思決定者は誰ですか。
