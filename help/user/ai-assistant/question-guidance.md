# Journey Optimizer B2B editionのAI アシスタントの質問ガイダンス

Journey Optimizer B2B editionのAI アシスタントに質問する場合は、次の質問の例を参照してください。 この情報には、AI アシスタントから最適な回答を得るために、質問にフレーズを付ける方法のヒントも含まれています。

## 客観的な質問

次の質問の例は、AI アシスタントを使用する際に達成できる目標に従ってグループ化されています。

| 目的 | 説明 | 例 |
| --- | --- | --- |
| 概念の学習とワークフローの継続 | 初心者でも、AI アシスタントを使用してReal-Time CDPとAdobe Journey Optimizer B2B editionの概念を学び、使い慣れない製品や機能を使いこなすことができます。 <br>経験豊富なユーザーは、AI アシスタントを使用して、ワークフローをブロックしている可能性のあるエッジケースを解決できます。 | <li>Real-Time CDPのユースケースを教えてください。 <li>購買グループのコンセプトを説明してもらいます。 |
| トラブルシューティング | AI アシスタントを使用して、ワークフローで発生する可能性のある基本的なエラーのデバッグ方法を学習します。 | <li>このエラー&lt;ERROR_MESSAGE>は何を意味しますか？ <li>「。..」という名前のオーディエンスを削除できないのはなぜですか？ |
| サンドボックスの健全性 | AI アシスタントを使用して、重複や未使用のオブジェクトを特定し、サンドボックスを効率的に管理できます。 | <li>類似したアカウントオーディエンスを見せてもらえますか？ <li>関連するデータセットを持たないスキーマはありますか？ |
| 価値分析 | AI アシスタントを使用して、最も使用されているデータオブジェクトを特定し、パフォーマンス指標を評価したり、最も価値のあるデータオブジェクトを見つけ出したりできます。 | <li>アカウント数は「。..」 セグメントの定義： <li>Experience Cloud Audiencesの宛先にオーディエンスがアクティブ化されたのはいつですか？ |
| 検索 | AI アシスタントを使用して、アカウントオーディエンス、データセット、デスティネーション、スキーマ、ソース、アカウントジャーニー、購買グループテンプレート、ソリューションの関心など、サポートされているExperience PlatformおよびAdobe Journey Optimizer B2B editionのオブジェクトを検索できます | <li>アカウントジャーニーで使用された名前に「Luma」を含むオーディエンスをリストします。 <li>XDM スキーマの「Luma: Custom Actions」にはどのような属性がありますか？ |
| 影響分析 | AI アシスタントを使用して、特定のワークフローで使用されたデータオブジェクトを特定し、変更の影響を評価できるようにします。 | <li>「B2B Person」スキーマで`workEmail.address`を使用するアカウントオーディエンスはどれですか？ <li>どのデータセットが… `jobTitle`に保存されていますか？ |

## 質問の文言

できるだけ正確な回答を得るには、AI アシスタントに質問を明確かつ文脈を持って伝える必要があります。 明確な質問をコンテキストに沿って行う方法のガイダンスについては、次のヒントのリストを参照してください。

* タスクや質問を簡潔に説明しましょう。
* あいまいな言語や複雑すぎる構文を避けて、理解を深めましょう。
* タスクや質問に関する関連するコンテキストを提供します。コンテキストは、AI アシスタントがより関連性の高い回答を生成するのに役立ちます。

次の表に、AI アシスタントを使用する際のベストプラクティスを示します。

| 推奨 | 例 |
| --- | --- |
| <li>取得または分析するオブジェクトまたは情報を具体的に指定します。 <li>データオブジェクト名を引用符で囲んでみます。 <li>オブジェクト名の一部しか知らない場合は、質問で指定することもできます。 | <li>「B2B アカウント」スキーマを使用するデータセット？ <li>名前に「アカウント」が含まれているアクティブなオーディエンスを表示します。 会員数でランク付けします。 |
| <li>あいまいな表現を避け、明確な言葉を使う： <li>正確な用語を使用して、クエリをより明確にします。 <li>Adobe Experience PlatformとAdobe Journey Optimizer B2B editionに関して質問する場合は、Experience PlatformまたはAdobe Journey Optimizer B2B editionに固有の用語を使用して、回答の関連性を高めてください。 | <li>「自分のアカウントオーディエンス」のメンバーは何人いますか？ <li>アカウントオーディエンス「マイアカウントオーディエンス」を使用するアカウントジャーニーの数は？ |
| <li>コンテキストを指定するか、条件を指定して結果をフィルタリングします。 <li>質問でフィルター条件を使用して、応答のデータ量を制限します。 | <li>アクティベートされておらず、6か月以上前に作成され、変更されていないアカウントオーディエンスを表示します。 <li>過去7日間に公開されたアカウントジャーニーを表示し、1,000人以上のメンバーを持つアカウントオーディエンスを使用します |

| 非推奨 | 例 |
| --- | --- |
| あいまいな言葉を使う： | <li>データセットに関する情報を受け取る。 <li>journey xは何をするものですか？ <li>「ACME Audience」には何人のユーザーがいますか？ <li>セグメントを表示。 <li>属性のリスト： |
| 不完全なリクエストを行う。 | <li>「Luma - Loyalty Dataset」 |
| コンテクストのない知識。 | <li>過去6か月間のオーディエンス。 <li>クエリを作成します。 <li>ジャーニーの作成 |
| 非常に複雑なクエリを作成する： | <li>あらゆるオブジェクトとその依存関係をまたいで、データリネージの包括的な分析を提供します。 |
| 基準やパラメーターを省略する。 | <li>データセットを表示する： |

## サポートされていない質問の例

次のリストは、Journey Optimizer B2B editionのAI アシスタントが現在サポートしていない質問の例を示しています。

* どのアカウントオーディエンスが、その条件で…フィールドグループのworkEmail.address フィールドを使用しますか？ 
* 配信ビジュアルで、10,000以上、5000から10,000未満、1000から5000未満のサイズのアカウントオーディエンスを使用したアクティブなジャーニーの数を表示します
* アカウントジャーニーxの最後の更新は誰でしたか？
* ソリューションの関心xに購買グループのメンバーを追加するアクティブなジャーニーの数
* ソリューションに関心を持つ購買グループのメンバーを追加するアクティブなジャーニーはどれですか？
* 購買グループテンプレートで最も一般的に使用される意思決定者の役職は何ですか？
* 購買グループテンプレート xの意思決定者は誰ですか？
