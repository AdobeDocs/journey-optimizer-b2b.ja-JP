# Journey Optimizer B2B Edition の AI アシスタントに関する質問ガイダンス

## 質問の言い回し

できるだけ正確な回答を得るために、AIアシスタントに明確さと文脈で質問を表現する必要があります。 コンテキストを使用して明確な質問をする方法のガイダンスについては、次のヒントのリストを参照してください。

* タスクや質問を簡潔に述べます。
* わかりやすくするために、あいまいな表現や過度に複雑な構文は避けてください。
* コンテキストは、AI Assistantがより関連性の高い応答を生成するのに役立つため、タスクや質問に関連するコンテキストを提供します。

次の表に、AI Assistant を使用する際にフォローするできるベストプラクティスをいくつか示します。

| する | 例 |
| --- | --- |
| <li>取得または分析するオブジェクトまたは情報を具体的に指定します。 <li>データオブジェクト名を引用符で囲んでみてください。 <li>オブジェクト名の一部しかわからない場合は、質問でそれを指定することもできます。 | <li>「B2Bアカウント」スキーマを使用しているデータセットはどれですか? <li>名前に「アカウント」を含むアクティブ化されたオーディエンス表示してください。 メンバー数でランク付けします。 |
| <li>あいまいさを避け、明確な言葉を使用してください。 <li>正確な用語を使用して、クエリをより明確にします。 <li>Adobe Experience Platform および Adobe Systems Journey Optimizer B2B Edition に関する質問を行う場合は、Experience Platform または Adobe Systems Journey Optimizer B2B Edition に固有の用語を使用して、回答の関連性を向上させるようにしてください。 | <li>「マイアカウントオーディエンス」には何人のメンバーがいますか? <li>アカウントオーディエンス「マイアカウントオーディエンス」を使用するアカウントジャーニーはいくつありますか? |
| <li>コンテキストを指定するか、結果をフィルタリングする条件を指定します。 <li>質問にフィルター条件を使用して、回答に含まれるデータの量を制限します。 | <li>表示アカウントアクティブ化されておらず、6か月以上前に作成され、変更されたことがないオーディエンスをします。 <li>表示過去 7 日間に公開されたジャーニーアカウント、1,000 人を超えるメンバーがいるアカウントオーディエンスを使用しています |

## サポートされていない質問の例

次のリストには、Journey Optimizer B2B エディションの AI アシスタントが現在サポートしていない質問の例が含まれています。

* どの アカウント オーディエンスが ...その状態のフィールドグループ?
* 配布ビジュアルで、10,000 人以上、5000 人から 10,000 人、1000 人から 5000 人、1000 人未満のサイズの アカウント 人の対象ユーザーを使用したアクティブなジャーニーの数表示
* アカウント ジャーニー xで最後の更新を行ったのは誰ですか?
* ソリューション興味 x で購入グループメンバーを追加するアクティブなジャーニーはいくつありますか?
* ソリューション興味xのためにメンバーグループ購入を追加するアクティブなジャーニーは何ですか?
* 購入グループテンプレートの最も一般的な意思決定者タイトルは何ですか？
* グループテンプレート x を購入するための意思決定者は誰ですか。
