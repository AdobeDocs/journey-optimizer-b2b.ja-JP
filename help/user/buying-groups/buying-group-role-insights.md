# ロールインサイトダッシュボード

ロールインサイトダッシュボードは、購買グループの役割が時間の経過と共にどのように進化し、関与するかを可視化します。 マーケターは、役割獲得のトレンド、エンゲージメントパターン、最近のキャンペーンが購入グループ内の様々な役割にわたるエンゲージメントをどのように促進しているかを理解するのに役立ちます。

このダッシュボードにアクセスするには、左側のナビゲーションで **[!UICONTROL アカウント]** を展開し、**[!UICONTROL 購入グループ]** を選択します。 「**[!UICONTROL 役割インサイト]**」タブを選択します。

![ ロールインサイトダッシュボード ](assets/roles-insights-dashboard.png){width="800" zoomable="yes"}

ダッシュボードには、次の 3 つのビューがあります。

| 表示 | 説明 |
| ---- | ----------- |
| [!UICONTROL  役割の獲得の履歴 ] | 購買グループに追加されたメンバー数の推移を、役割別および月別にセグメント化して視覚化します。 |
| [!UICONTROL  役割別エンゲージメント ] | 購入グループロール別に、エンゲージメントアクティビティの合計数（メール開封数、クリック数、フォーム送信数など）を表示します。 |
| [!UICONTROL  役割ごとのエンゲージメントの推移 ] | 購買グループのエンゲージメントアクティビティを経時的に追跡し、役割別の毎月のエンゲージメントトレンドを表示します。 |

## データのフィルタリング

左上の _フィルター_ （![ フィルターアイコン ](../assets/do-not-localize/icon-filter.svg)）アイコンをクリックし、次の属性のいずれかを使用して表示データをフィルタリングします。

* **[!UICONTROL 役割]** – 選択した 1 つ以上の購入グループの役割でデータをフィルタリングします。
* **[!UICONTROL ソリューションの関心]** - 1 つ以上の選択されたソリューションの関心でデータをフィルタリングし、特定の製品ラインや製品に焦点を当てます。
* **[!UICONTROL 日付範囲]** - データを特定の期間に制限します（デフォルトは過去 1 年）。

![ 役割インサイトのフィルターダイアログ ](./assets/roles-insights-filter-dialog.png){width="400"}

各属性について、データのフィルタに使用する値を選択し、「**[!UICONTROL 適用]**」をクリックします。

## [!UICONTROL  役割の獲得の履歴 ]

マーケティングチームやセールスチームが新しい人物を獲得し、エンゲージメントを促進し、データを強化すると、購入グループの新しいメンバーが資格を獲得します。 このレポートでは、これらの取り組みが、新たに認定された購買グループのメンバーにどのように変換されるかを、役割別に分類して示しています。 ユーザーが購買グループに追加されるたびに、レポートは対応する役割の下にその追加を記録します。

![ ロールインサイトダッシュボード – 時間の経過に伴う役割の獲得グラフ ](assets/roles-insights-role-aquisition-over-time.png){width="600" zoomable="yes"}

グラフの各行は、役割を表します。 線上のプロットポイントにカーソルを合わせると、次のような詳細が表示されます。

* 取得日
* Count of acquisitions

詳細情報を表示するには、右上の「**...**」メニューアイコンをクリックします。

## [!UICONTROL  役割別エンゲージメント ]

このレポートは、選択した期間の購買グループ役割別のエンゲージメントアクティビティ量の合計を要約します。 このレポートを使用して、各ロールがマーケティング活動にどのように対応しているかを把握します。

![ 役割インサイトダッシュボード – 役割別エンゲージメントグラフ ](assets/roles-insights-engagement-by-role.png){width="500" zoomable="yes"}

グラフの各棒グラフは、役割を表します。 バーの上にマウスポインターを置くと、表示されるカウントの詳細が表示されます。以下に例を示します。

* 役割名
* エンゲージメントカウント

詳細情報を表示するには、右上の「**...**」メニューアイコンをクリックします。

## [!UICONTROL  役割ごとのエンゲージメントの推移 ]

このレポートは、時間の経過と共に購入グループアクティビティを追跡し、最近のキャンペーンが様々な役割にわたるエンゲージメントをどのように促進しているかを理解するのに役立ちます。 この情報を使用して、マーケティング戦略を最適化し、特定の役割をより効果的にターゲットに設定します。

![ 役割インサイトダッシュボード – 役割別のエンゲージメントの推移グラフ ](assets/roles-insights-engagement-by-role-over-time.png){width="500" zoomable="yes"}

グラフの各行は、役割を表します。 線上のプロットポイントにカーソルを合わせると、次のような詳細が表示されます。

* 日付
* エンゲージメントカウント

詳細情報を表示するには、右上の「**...**」メニューアイコンをクリックします。

## データの操作

データを操作するには、各グラフの右上にある _その他_ （**...**）メニューを使用します。

### [!UICONTROL  ドリルスルー ]

_[!UICONTROL 役割別のエンゲージメント]_ については、**[!UICONTROL ドリルスルー]** を選択すると、役割およびソリューションの興味ごとにエンゲージメントを詳細に分析できます。

ダッシュボードに適用されたグローバルフィルターは引き継がれます。 左上の _フィルター_ （![ フィルターアイコン ](../assets/do-not-localize/icon-filter.svg)）アイコンをクリックし、ドリルスルー表示の [ 属性フィルターの変更 ](#filter-the-data) をクリックします。

### [!UICONTROL  詳細を表示 ]

**[!UICONTROL さらに表示]** を選択すると、拡張されたデータとインサイトが表示されます。

表示されるポップアップには、グラフと、データの分類を表示するテーブルが含まれます。
