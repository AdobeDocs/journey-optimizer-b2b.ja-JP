# Web エンゲージメントダッシュボード

Web エンゲージメントダッシュボードでは、web 訪問者が主要コンテンツとやり取りする方法を可視化します。 アカウントの業界や地域をまたいだデータをセグメント化し、エンゲージメントのトレンドを理解するのに役立ちます。 このダッシュボードを使用すると、コンテンツ戦略とアカウントのターゲティングを示す web 行動パターンを表示して、戦略的な意思決定をサポートできます。

_Web エンゲージメントダッシュボード_ にアクセスするには、左側のナビゲーションで **[!UICONTROL ダッシュボード]** 項目を選択します。 次に、ページ上部の「**[!UICONTROL Web エンゲージメント]**」タブを選択します。

![Web エンゲージメントダッシュボード &#x200B;](./assets/web-engagement-dashboard.png){width="700" zoomable="yes"}

## データのフィルタリング

左上の _フィルター_ （![&#x200B; フィルターアイコン &#x200B;](../assets/do-not-localize/icon-filter.svg)）アイコンをクリックし、次の属性のいずれかを使用して表示されたデータをフィルタリングします。

* **[!UICONTROL アカウント地域]** - アカウントに関連付けられた、選択した 1 つ以上の地理的地域でデータをフィルタリングします。
* **[!UICONTROL アカウント業界]** - アカウントに関連付けられた、選択した 1 つ以上の業界の分類でデータをフィルタリングします。
* **[!UICONTROL 日付範囲]** – 選択した日付範囲でデータをフィルタリングします。 デフォルトの範囲は現在の日付です。

![&#x200B; 表示されたデータを属性でフィルタリング &#x200B;](./assets/web-engagement-dashboard-filters-dialog.png){width="500"}

データのフィルタリングに使用する各属性の値を選択し、「**[!UICONTROL 適用]**」をクリックします。

## [!UICONTROL &#x200B; 上位のページビュー &#x200B;] {#top-page-views}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_top_page_views"
>title="上位ページビュー数"
>abstract="Web サイト上で最も頻繁に表示されるページで、最もトラフィックが多いコンテンツを特定するのに役立ちます。"

この表には、最も閲覧された web ページの上位 10 位が表示され、訪問者から最も共感を得られるコンテンツを特定するのに役立ちます。 データには、次のものが含まれます。

| 列 | 説明 |
| ------ | ----------- |
| ページ名 | Web ページの名前またはタイトル。 |
| 合計表示数 | ページが表示された合計数。 |
| 既知訪問者（%） | 既知の（識別された）訪問者に起因するページビューの割合。 |
| 不明な訪問者（%） | 不明な（匿名の）訪問者に起因するページビューの割合。 |

![&#x200B; 上位ページビューテーブル &#x200B;](./assets/web-engagement-dashboard-top-page-views.png){width="650" zoomable="yes"}

## [!UICONTROL &#x200B; アカウント地域別のページビュー数 &#x200B;] {#page-views-by-region}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_page_views_by_region"
>title="アカウント地域別ページビュー数"
>abstract="関連するアカウントの地理的地域によってセグメント化された web 訪問者分布。"

このビジュアライゼーションでは、訪問者数がアカウント地域別にセグメント化されて表示されます。 この例では、web トラフィックが様々な地域でどのように異なるかを示し、地域のオーディエンスに合わせてコンテンツやキャンペーンを調整できるようにします。 グラフの棒の上にマウスポインターを置くと、次のような詳細が表示されます。

* アカウント地域の名前
* ページビュー数

![&#x200B; アカウント地域別のページビュー数 &#x200B;](./assets/web-engagement-dashboard-page-views-by-account-region.png){width="500" zoomable="yes"}

## [!UICONTROL &#x200B; アカウント別ページビュー数 &#x200B;] {#page-views-by-industry}

>[!CONTEXTUALHELP]

>title="アカウント業種別ページビュー数"

このビジュアライゼーションには、アカウント業界ごとにセグメント化された訪問者数が表示されます。 このグラフを使用すると、web トラフィックが様々な業界でどのように異なるかを把握し、業界固有のコンテンツ戦略を策定できるようになります。 グラフの棒の上にマウスポインターを置くと、次のような詳細が表示されます。

* アカウントの業界名
* ページビュー数

![&#x200B; アカウント別ページビュー数 &#x200B;](./assets/web-engagement-dashboard-page-views-by-account-industry.png){width="500" zoomable="yes"}

## データの操作

データを操作するには、各グラフの右上にある _その他_ （**...**）メニューを使用し、「**[!UICONTROL その他を表示]**」を選択して、拡張されたデータとインサイトを表示します。

表示されるポップアップには、グラフと、データの分類を表示するテーブルが含まれます。

データをダウンロードするには、データテーブルの右上にある **[!UICONTROL CSV をダウンロード]** をクリックします。

![&#x200B; トップページビュー表その他のポップアップを表示 – 「CSV をダウンロード」をクリック &#x200B;](./assets/web-engagement-dashboard-top-page-views-view-more.png){width="700" zoomable="yes"}
